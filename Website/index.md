## 简介

**KFACBT's Toolkit** 是 `KFACBT` 使用 `易语言` 开发的一款面向 `Windows` 的工具箱。

- 技术关键词: 易语言 / EPL / 工具箱
- 支持的操作系统: Windows 7 - Windows 11

## 功能

目前，该工具箱含有以下功能。

- 时钟
- 蜂鸣器
- 桌面/分辨率管理
- 播放器
- 网域管理
- 朗读功能
- 系统管理功能

>其他功能可以下载程序或编译源代码查看。

## 编译

该软件采用易语言编写，如果你想要编译该软件，请准备以下环境：

- Windows 7 和以上版本的电脑
- 易语言 5.93

同时，该软件引用了部分 [精易模块](https://ec.125.la/) 的代码，请在编辑/引用代码时遵守相关协议。
